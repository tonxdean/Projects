Summary: Create word search puzzles with conditions specified in the input files

The program expects three command line arguments, the name of the input file, the name of the
output file, and a string that tells the program to either form only one solution, or all solutions:

inverse_word_search.exe puzzle2.txt out2.txt one_solution
inverse_word_search.exe puzzle2.txt out2.txt all_solutions
